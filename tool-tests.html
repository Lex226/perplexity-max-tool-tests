Agent API
API

sonar-deep-research через REST API. Async-режим для долгих запросов, tool calls, Python SDK. Для автоматизации рабочих процессов.

Код — Python async Deep Research
import os from perplexity import Perplexity client = Perplexity(api_key=os.environ["PPLX_API_KEY"]) # Async Deep Research — занимает 2-4 минуты response = client.responses.create( preset="advanced-deep-research", input="Лучшие практики партиционирования таблиц в SQL Server 2019 для хранилища данных с ежедневной загрузкой 500к строк" ) print(response.output_text) # Количество поисковых запросов: print(f"Searches: {response.usage.num_search_queries}")
Документация: docs.perplexity.ai/docs/agent-api. Получи API ключ в Settings → API.
